# Taiko Documentation [Guides, protocol specs, and network reference for Taiko, a based rollup on Ethereum. Deploy contracts, bridge tokens, and run a node.]

## Welcome to Taiko!

Taiko is an Ethereum-equivalent (type-1) based rollup. "Based" means Ethereum L1 validators sequence Taiko blocks — there is no centralized sequencer. Standard EVM tooling works without modification.

These docs cover everything from deploying your first contract to running a node and understanding the protocol.
