# Receive Tokens [Get the testnet assets you need for Taiko Hoodi.]

## Prerequisites

- A wallet such as MetaMask or Rabby
- Taiko Hoodi added to your wallet. See [Connect to Taiko](/quickstart/connect).

## Get Ethereum Hoodi ETH

Taiko Hoodi settles to the Ethereum Hoodi testnet, so start by funding your L1 wallet with Hoodi ETH from a public faucet.

Once you have Hoodi ETH on L1, you can bridge it to Taiko Hoodi using the [Hoodi bridge](https://bridge.hoodi.taiko.xyz).

## Get Test Tokens

For test ERC-20 flows, Taiko Hoodi also exposes a faucet flow through the bridge UI.

At the time of writing, the common test token is:

- [HORSE](https://hoodi.etherscan.io/token/0x0a5db5597adc81c871ebd89e81cfa07bdc8fafe3)

Open the [Hoodi bridge](https://bridge.hoodi.taiko.xyz), use the faucet option in the sidebar, and mint test tokens on Ethereum Hoodi before bridging them to Taiko Hoodi.

## Bridge Assets to Taiko Hoodi

Once you have L1 Hoodi ETH or test tokens:

1. Open the [Hoodi bridge](https://bridge.hoodi.taiko.xyz)
2. Connect your wallet
3. Select the asset you want to move from Ethereum Hoodi to Taiko Hoodi
4. Submit the bridge transaction

For a contract-level walkthrough, see [Bridge Tokens](/guides/bridge-tokens).

## Notes

- Taiko **mainnet** does not have a faucet flow. This page is for **testnet usage only**.
- If you only need ETH on Taiko Hoodi, bridging Hoodi ETH is the simplest path.
