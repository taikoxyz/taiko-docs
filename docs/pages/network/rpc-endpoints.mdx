# RPC Endpoints

RPC endpoints, chain IDs, and explorer URLs for all Taiko networks.

## Taiko Mainnet (Alethia)

| Field             | Value                                |
| ----------------- | ------------------------------------ |
| Network Name      | Taiko Mainnet                        |
| Chain ID          | `167000`                             |
| Chain ID (Hex)    | `0x28C58`                            |
| RPC URL           | `https://rpc.mainnet.taiko.xyz`      |
| Explorer          | `https://taikoscan.io`               |
| Explorer API      | `https://api.taikoscan.io/api`       |
| Bridge            | `https://bridge.taiko.xyz`           |
| Native Currency   | ETH                                  |

## L1 Reference Chains

| Network            | Chain ID  |
| ------------------ | --------- |
| Ethereum Mainnet   | `1`       |
