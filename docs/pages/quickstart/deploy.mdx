# Deploy a Contract

Deploy and verify your first smart contract on Taiko.

:::code-group

```txt [Agent]
Read https://docs.taiko.xyz/SKILL.md and deploy my contract to Taiko mainnet
```

```txt [Human]
Follow the step-by-step guide below.
```

:::

:::warning
Taiko runs the **Shanghai** EVM. Solidity 0.8.24+ defaults to Cancun opcodes (`MCOPY`, `TSTORE`, etc.) that do not exist on Taiko. Set `evm_version = "shanghai"` or your deployment will revert.
:::

## Prerequisites

- [Foundry](https://book.getfoundry.sh/getting-started/installation) installed
- ETH on Taiko (bridge from Ethereum L1)
- [Taikoscan](https://taikoscan.io) API key for verification

## Deploy

::::steps

### Configure Foundry

Add a Taiko profile to `foundry.toml`:

```toml
[profile.taiko]
evm_version = "shanghai"
```

### Build

```bash
FOUNDRY_PROFILE=taiko forge build
```

### Deploy and verify

:::code-group

```bash [Mainnet]
FOUNDRY_PROFILE=taiko forge create src/Counter.sol:Counter \
  --rpc-url https://rpc.mainnet.taiko.xyz \
  --private-key $PRIVATE_KEY \
  --verify \
  --verifier etherscan \
  --verifier-url https://api.taikoscan.io/api \
  --etherscan-api-key $TAIKOSCAN_API_KEY
```

```bash [Hoodi Testnet]
FOUNDRY_PROFILE=taiko forge create src/Counter.sol:Counter \
  --rpc-url https://rpc.hoodi.taiko.xyz \
  --private-key $PRIVATE_KEY \
  --verify \
  --verifier etherscan \
  --verifier-url https://api-hoodi.taikoscan.io/api \
  --etherscan-api-key $TAIKOSCAN_API_KEY
```

:::

### Confirm deployment

```bash
cast code $ADDRESS --rpc-url https://rpc.mainnet.taiko.xyz
```

A non-empty hex response confirms the contract is deployed.
